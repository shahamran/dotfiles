# @ransh config files

This repo uses [chezmoi] to manage the configuration files.

[chezmoi]: https://www.chezmoi.io/

## Dependencies

* Terminal: [WezTerm]
* Shell: fish
* Prompt: [starship]
* Editor: [Neovim]

[WezTerm]: https://wezfurlong.org/wezterm/
[starship]: https://starship.rs/
[Neovim]: https://neovim.io/
